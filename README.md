# FSA Image Editor

Ferramenta feita em Java para tratamento de imagens, a partir de operações e filtros.

## Operações Aritméticas
- Soma de Imagem/Cor
- Subtração de Imagem/Cor
- Multiplicação de Imagem/Cor
- Divisão de Imagem/Cor
- Blending
